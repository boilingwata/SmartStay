# Tech Stack

## Core Framework
- **React 18**: Frontend library for building the user interface.
- **Vite**: Modern build tool and development server.
- **TypeScript**: Static typing for enhanced developer experience and code quality.

## Data Management
- **Supabase**: Backend-as-a-Service providing PostgreSQL (with `smartstay` schema), Authentication, and Storage.
- **TanStack Query (v5)**: Powerful asynchronous state management for data fetching and caching.
- **Zustand**: Lightweight state management for global application state (Auth, UI, Notifications).

## UI & Styling
- **Tailwind CSS**: Utility-first CSS framework for rapid UI development.
- **Shadcn UI**: Collection of re-usable components built with Radix UI and Tailwind.
- **Lucide React**: Consistent and beautiful icon library.
- **Framer Motion**: Library for production-ready animations.

## Forms & Validation
- **React Hook Form**: Performant, flexible, and extensible forms.
- **Zod**: TypeScript-first schema declaration and validation library.

## Internationalization
- **i18next / react-i18next**: Framework for localization (Vietnamese and English).
