# Integrations

## Backend Services
- **Supabase**:
    - **PostgreSQL**: Primary database using the `smartstay` schema.
    - **Auth**: Handles user registration, login, and session management.
    - **Storage**: Used for uploading contracts, receipts, and user avatars.

## Monitoring & Analytics
- **Sentry**: Error tracking and performance monitoring integration.

## Content & UI Utilities
- **TipTap**: Headless rich text editor for announcements and descriptions.
- **Recharts**: Composible charting library for dashboard analytics.
- **dnd-kit**: Drag-and-drop toolkit for room management and task boards.
- **Lucide React**: Icon set for the entire application.

## Document Export
- **jsPDF**: Client-side PDF generation for invoices and contracts.
- **html2canvas**: Screenshot/canvas capture for converting HTML elements to images for PDF export.

## Others
- **date-fns**: Modern JavaScript date utility library.
- **qrcode.react**: QR code generation for payment and identification.
- **react-confetti**: Visual feedback for successful actions.
